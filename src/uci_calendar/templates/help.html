📚 Help & Instructions

📱 How to Subscribe

It's easy! Just add the calendar URL below to your favorite calendar app:

https://petecog.github.io/cycle-calendar/calendar.ics
📥 Download Calendar 📲 Subscribe (webcal)

Platform-specific instructions:

  • Google Calendar: On computer, go to Settings → Add calendar → From URL. Paste the link.
  • Apple Calendar: On computer, go to File → New Calendar Subscription. Paste the link.
  • Outlook: Click "Add calendar" → Subscribe from web. Paste the link.
  • Phone apps: Most apps have an "Add calendar" or "Subscribe" option in settings.
🤖 Automatic Updates: This calendar syncs with UCI weekly - you'll never have to manually update it!

🔧 Not seeing events?

If your calendar seems empty:

  • Wait a few minutes - Some apps take time to load new subscriptions
  • Check the debug view to see if events are being found
  • Try the webcal:// link above - some phones prefer this format
  • Download the .ics file as a backup and import it manually

🔄 How it works

Every week, this system:

  1. Downloads the latest race schedules from UCI's official website
  2. Converts them into calendar format that any app can read
  3. Updates the calendar file automatically

It's completely automated, so you get new races and schedule changes without doing anything.

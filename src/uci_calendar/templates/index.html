ü§î What is this?

The UCI (Union Cycliste Internationale) is cycling's world governing body. They organize all the major mountain bike competitions - World Cups, World Championships, and continental events. Normally you'd have to manually check their website to see when races are happening.

This service automatically watches the UCI calendar and converts it into a format your calendar app can understand. Once you subscribe, you'll get automatic updates whenever new races are announced or schedules change.

üîÑ How it works

Every week, this system:

  1. Downloads the latest race schedules from UCI's official website
  2. Converts them into calendar format that any app can read
  3. Updates the calendar file automatically

It's completely automated, so you get new races and schedule changes without doing anything.
